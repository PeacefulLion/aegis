# Aegis
基于 badjs 的前端监控体系平台

## 开发 
语言 ts
构建 parcel
UI框架 echarts + ant-design
架构 react hook 

## whistle 配置
```
//badjs2.ivweb.io/badjs3/ localhost:1234/
/^https?://badjs2\.ivweb\.io/(.*\.(js|css|png|jpg|gif|jpeg|svg|blob).*)$/ localhost:1234/$1
```

## 调试
1. ``` npm start ```

2. 登录 badjs2.ivweb.io 获取登录态

3. 打开 badjs2.ivweb.io/badjs3 开始调试
